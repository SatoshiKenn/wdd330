Challenge Proposal

    Purpose

    The purpose of this site is for pokemon players. As a pokemon player, I know when we are starting a new game or looking for different teams that have more competitive we always look for information on the internet to know the specifications for each pokemon we want in our team, something I realized is that is for sure that we will looking for the pokemons more than one time and we need to go find again the pokemon we want to see, that is the main problem I want to solve, give an option to save the 6 favorite pokemon of the user so they can access faster to the pokemons specifications.

    Audience

    The audience fot this project is everyone who wants to look for pokemon information, to know and plan a team for a pokemon game they are currently playing, as a Pokemon player I know how usually look for this kind of information, I know it will be a useful tool for pokemon players.

    Data sources

    I'll use the Pokemon API to obtain all the information to create the Pokemon cards and localStorage to save maximum 6 favorite pokemon.
